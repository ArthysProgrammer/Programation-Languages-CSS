O CSS é uma linguagem que faz parte do trio HTML, CSS e Java Script. Sua função é "estilizar" o texto, deixando-o mais bonito visualmente!

Seu nome significa "Cascading Style Sheets", que traduzindo para português fica "Folhas de Estilo em Cascata"!

Na minha opnião é uma linguagem muito fácil de aprender junto com seus amigos(HTML e Java Script) e muito divertida e útil!
